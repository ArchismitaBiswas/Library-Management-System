# Library Management System

## Overview

A Python-based Library Management System using SQLite and Pandas to manage books, students, and issue/return records.

## Features

* Add and manage books
* Manage student records
* Track issued books
* Track returned books
* Generate library reports
* SQLite database integration

## Technologies Used

* Python
* SQLite
* Pandas

## Database Tables

* Books
* Students
* IssueRecords

## Output

* Library Database
* Library Report
* Issue Tracking System

## Conclusion

This project demonstrates database management, CRUD operations, SQLite integration, and real-world application development using Python.
